Lua Converter CLI Tool
======================

Usage:
  ./lua_converter.sh <input.java|input.js|input.jar>

Requirements:
- Java 11+ (OpenJDK or Oracle)
- lua_converter_tool_main.jar (place in same directory)
- Works on Windows (Git Bash), Linux, macOS

Output:
- <input>.lua generated in current directory

Supports Java, JS, JAR input files. All conversion logic and dependencies are embedded in the JAR.
